# Commercial Release Checklist

Before listing any pattern publicly:
- Make one complete sample in the stated yarn weight.
- Confirm gauge before and after blocking.
- Verify every stitch count at all shaping points.
- Photograph construction steps that may be ambiguous.
- Test every garment size with at least one independent maker where practical.
- Run accessibility review: define abbreviations, avoid color-only directions, and include written alternatives to charts.
- Check that finished measurements include intended ease.
- Confirm yarn quantities with a 10-15 percent safety margin.
- Tech edit spelling, numbering, repeats, and cross-references.
- Add final graphics only after the written pattern is locked.
