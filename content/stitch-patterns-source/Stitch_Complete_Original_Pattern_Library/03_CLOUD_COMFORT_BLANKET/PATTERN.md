# Cloud Comfort Blanket

A soft neutral blanket alternating mesh, puff, and ribbed texture bands.

## Materials
Bulky-weight yarn:
- Baby: 1,150 yd
- Throw: 2,050 yd
- Large: 2,900 yd
8 mm hook, or size needed for gauge.
Tapestry needle.

## Gauge
10 hdc = 4 in; 7 hdc rows = 4 in after washing.

## Foundation counts
The pattern multiple is 6 + 3.
- Baby: ch 93
- Throw: ch 129
- Large: ch 153

## Special stitches
**Puff:** Yarn over, insert hook, pull up a loop to dc height 4 times in same stitch, yarn over and pull through 8 loops, yarn over and pull through final 2 loops.

**Front-post dc:** Yarn over, insert hook from front to back to front around post of indicated stitch, complete dc.

## Main pattern

### Setup
Row 1: Hdc in third ch from hook and every chain across. Turn.
Stitch counts: 91 (127, 151).

### Texture repeat - 12 rows
Row 2: Ch 2, hdc BLO across. Turn.
Row 3: Ch 2, hdc across. Turn.
Row 4: Ch 3, *skip 2 sts, puff in next st, ch 2; repeat from * to last 1 st, dc in last st. Turn.
Row 5: Ch 2, *2 hdc in ch-2 space, hdc in top of puff; repeat across, hdc in final turning chain. Turn.
Row 6: Ch 3, dc across. Turn.
Row 7: Ch 3, *front-post dc around next st, back-post dc around next st; repeat across, dc in last st. Turn.
Row 8: Repeat Row 7, working post stitches as they appear.
Rows 9-10: Ch 2, hdc BLO across. Turn.
Row 11: Ch 1, sc across. Turn.
Row 12: Ch 3, *skip 2 sts, 5 dc in next st, skip 2 sts, sc in next st; repeat across, adjusting final repeat to end with sc. Turn.
Row 13: Ch 2, hdc in each stitch across, working one hdc in each shell stitch and sc. Turn.

Repeat Rows 2-13 until blanket measures approximately:
- Baby: 40 in before border
- Throw: 58 in before border
- Large: 70 in before border

End after Row 3, 6, 9, 10, 11, or 13 so the top edge is straight.

## Border
Round 1: Ch 1, sc evenly around. Work 3 sc in each corner. Along side edges use approximately 2 sc per hdc-row pair and 3 sc per two dc rows. Join.
Round 2: Ch 1, hdc BLO around, placing 3 hdc in each corner stitch. Join.
Round 3: Ch 1, *sc, ch 2, skip 1; repeat around, adjusting at corners with (sc, ch 2, sc). Join.
Round 4: Sl st into first ch-2 space. Work 3 hdc in each ch-2 space around; in each corner space work 5 hdc. Join.
Fasten off.

## Finishing
Weave ends through matching texture bands. Wash according to yarn label and dry flat or tumble low only if the yarn permits.

## Custom sizing
Each 6-chain repeat adds approximately 2.4 in. Add or remove repeats while preserving the multiple of 6 + 3.

## Troubleshooting
**Puff rows narrow:** Relax the chain-2 spaces or use a hook 0.5 mm larger only for puff rows.
**Shell row flares:** Skip an extra stitch in a few evenly spaced repeats.
**Side edge uneven:** Count row repeats and place border stitches consistently.
