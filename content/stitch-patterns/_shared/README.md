# Stitch Complete Original Pattern Library

This pack contains twelve original project patterns based on the project concepts visible in the Stitch homepage mockup. The written instructions are newly authored for Stitch; they are not copied from or reverse-engineered from a third-party pattern.

## Included for every project
- Full materials and notions
- Finished measurements
- Gauge and swatching instructions
- Abbreviations and special stitches
- Detailed start-to-finish instructions
- Assembly and finishing
- Customization guidance
- Care instructions
- Troubleshooting
- Accessibility-oriented plain-language notes
- Cursor-ready `pattern.json`
- Maker checklist

## Important production note
The stitch arithmetic and construction logic have been checked during authoring. Physical yarn, personal tension, fiber behavior, and body measurements vary. Wearables should still be sample-made and size-tested before being sold as final commercial patterns. The pattern pages tell makers to measure and block rather than relying only on nominal sizing.

## Folder use in Cursor
Each project folder contains human-readable Markdown and structured JSON. Cursor can map:
- `title`, `craft`, `difficulty`, `sizes`, and `tags` to pattern cards
- `materials` and `gauge` to preparation panels
- `sections` to step-by-step mode
- `checklist.md` to the project tracker
- `image_prompt.txt` to future graphics generation
